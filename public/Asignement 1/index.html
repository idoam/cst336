Presentation

Welcome to the Vim Fan page, here you will (maybe) discover vim and all its advantages!

Vim is a very complete terminal visual text editor, very famous because it is the successor of "vi", included with UNIX based systems. See more in history.

It is very powerful, customizable and open source. Contrary to recent text editors, it must be trained to be used efficiently.

It is mainly designed for keyboard users, but there is a version compatible with the mouse (see customization).


How to use

Vim contains 6 main modes, but the two most used are the normal mode and the insert mode.

While you are in the normal mode, you can't write, but you can still interact with your file threw commands.

Pressing i, a or o puts you into the insert mode (at different places around your cursor). In this mode, you can write in your file.


See more in the tutorial section !

Why to use

You should use vim because it is a really powerfull tool, compatible with a lot of languages.

Programming using a text editor permits to keep a clear architecture.

Compiling using the console line commands permits to gain deeper knowledge about :

  1. the language you are working on
  2. your systems'architecture
  3. the bash that you use
